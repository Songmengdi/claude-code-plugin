# 阶段 4：文档生成

前提：阶段 3 已完成，文档结构已确认。

## 目标

按确认的结构生成完整文档。

## 示例规则

每个示例必须是**生产可用的最小闭环**：
- 明确初始状态
- 完整调用
- 明确返回结果
- 明确状态变化

**禁止**：伪代码、不完整 JSON、未定义的占位符

## 参数规则

每项参数必须明确（附 source）：
- 是否必填
- 类型
- 默认值
- 对行为的影响

## 错误规则

- 错误、约束必须是第一类内容，与接口或示例同级
- 每项必须明确（附 source）：
  - 触发条件
  - 后果
  - 是否可恢复

## 本阶段输出

- 所有文档文件已生成
- 每个事实性陈述已附 source 标记
- index.md 链接已指向实际文档

## 完成条件

全部文档生成完毕，用户确认后进入阶段 5。
