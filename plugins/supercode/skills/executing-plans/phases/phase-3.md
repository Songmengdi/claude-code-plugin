# 阶段 3：汇报

**前提：阶段 2 已完成，批次任务执行完毕。**

## 目标

向用户汇报批次执行情况。

## 执行方式

### 汇报内容

1. **已实现的内容**
   - 列出本批次完成的主要功能

2. **测试验证输出**
   - 展示按计划中测试方式验证的输出
   - 单元测试结果
   - 集成测试结果（如有）
   - 用户协助验证状态（如有）

3. **接收反馈**
   - 说："准备接收反馈。"
   - 等待用户回应
