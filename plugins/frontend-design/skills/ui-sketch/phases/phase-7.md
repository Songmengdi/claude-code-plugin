# 阶段 7：展示初稿与迭代

## 目标

向用户展示草图，收集反馈并迭代修改。

## 执行

1. **展示初稿** — 告知用户草图已完成，等待反馈
2. **迭代修改** — 根据用户反馈使用工具调整：
   - 修改元素：`update_element`
   - 删除元素：`delete_element`
   - 新增元素：`create_element` / `batch_create_elements`

## 迭代原则

- 每次只处理用户明确提出的修改点
- 修改后重新检查坐标，确保无覆盖
- 若修改涉及布局变化，需重新计算坐标

## 完成条件

用户确认草图无需再改，进入阶段 8（保存）。
