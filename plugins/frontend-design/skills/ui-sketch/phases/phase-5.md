# 阶段 5：绘制说明卡片

## 目标

为每个页面（PC 或 Phone）绘制说明卡片。

**读取本文件后，你才知道说明卡片的具体规范。**

## 说明卡片规范

| 属性 | PC 说明 | Phone 说明 |
|------|---------|------------|
| 宽度 | 280px | 280px |
| 边框色 | `#4ecdc4`（青绿） | `#ff6b6b`（红色） |
| 标题 | "页面说明" | "移动端说明" |
| 位置 | 页面右侧 100px | 页面右侧 100px |

## 必填内容

**只写草图无法表达的信息**，避免重复：

1. **页面尺寸** — 如 "1920 × 1080 (1080p)"
2. **布局行为** — 草图画的框是固定的还是可滚动的？哪些区域吸顶/吸底？
3. **设计风格** — 为后续精细化设计定调（极简/商务/活泼/暗黑…），不是颜色 specification
4. **交互说明** — 草图中虚线框、箭头等交互元素的具体行为：
   - 弹出层：点击什么触发、显示什么内容、如何关闭
   - 下拉菜单：包含哪些选项
   - 导航行为：跳转到哪个页面
   - 手势操作：左滑/右滑/下拉触发什么
5. **状态说明** — 指向下方变体行，列出已展示和未展示的状态，未展示的用文字简要描述

## 示例

```
┌─────────────────────────────┐
│ 页面说明                      │
│ ─────────────────────────── │
│ 尺寸: 1920 × 1080            │
│                             │
│ 布局行为                     │
│ • 导航栏吸顶固定              │
│ • 主内容区垂直滚动             │
│ • 右侧边栏固定，不随内容滚动   │
│                             │
│ 设计风格                     │
│ 极简留白，信息密度低           │
│                             │
│ 交互说明                     │
│ • 点击汉堡菜单→展开侧边导航    │
│ • 点击分类标签→筛选文章列表    │
│ • 文章卡片→进入文章详情页      │
│                             │
│ 未展示状态                   │
│ • 空状态：无文章时显示引导      │
│ • 加载态：骨架屏              │
│ （hover/disabled 见下方变体行） │
└─────────────────────────────┘
```

## 完成条件

说明卡片绘制完成。若有变体需求，进入阶段 6；否则直接进入阶段 7。
