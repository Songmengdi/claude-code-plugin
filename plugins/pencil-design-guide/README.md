# Pencil Design Guide

Pencil MCP 工具使用指南技能插件。

## 功能

提供使用 Pencil MCP 工具进行 Figma 设计的最佳实践指导，包括：

- 核心原则（读取→修改→验证、分批操作、及时验证）
- 避免失败的 5 个关键规则
- 操作流程模板
- 快速检查清单
- 图标设置和对齐速查

## 安装

插件已安装在全局插件目录，自动可用。

## 使用

当以下场景时自动触发：

- 需要进行精细化的 UI 设计
- 询问 Pencil 工具相关问题
- 提到 Figma 设计或批量操作设计节点
- 准备调用 Pencil MCP 工具时
